# 📊 Pharma-Metrics

Pharmaceutical Sales Analysis: Transforming wholesale-retail sales data into actionable business insights via interactive Power BI dashboards.

---

## 🚀 Key Highlights

* **Exploratory Data Analysis (EDA):** Employed Python (Pandas) to inspect data quality, handle anomalies, and analyze sales trends.
* **Data Modeling:** Designed a robust star schema data model in Power BI's Query Editor for efficient query performance and streamlined data retrieval.
* **Interactive Dashboards:** Created two interactive Power BI dashboards:
  * **Sales Overview Snapshot:** Displays high-level sales KPIs, trends, and team performances.
  * **In-Depth Business Analytics:** Offers granular insights from distributor and customer perspectives.

---
